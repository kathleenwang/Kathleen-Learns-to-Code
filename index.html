    200 coding projects in 200 days

    Inspired by Jennifer Dewalt's 180 websites in 180 days , I decided to also learn coding this way! I originally wanted to do 200 games, but with little to no coding experience, I figured I should start with websites. Though if throughout this experience I feel more comfortable with coding other things, I may do so.

    My rules

    • Whatever work is done for the day will be published (regardless of whether it's finished or not).
    • A description of the day's project will be posted on my blog.
    • A list of coding techniques/methods will be created (more for personal note to see what I've learned so far).
